\# CNN Velocity Reconstruction



This repository contains cnn codes for reconstructing halo line-of-sight

peculiar velocities using convolutional neural networks applied to

CAMELS IllustrisTNG simulations.



\## inputs tested

\- Density

\- Density + position

\- Density + position + halo mass

\- Position 

\- Halo mass

\- Position + halo mass



\## aim

create a velocity reconstruction code using CNN and compare

CNN performance against linear theory velocity reconstruction.
